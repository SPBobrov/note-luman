Prompt-book

 Используя html css js и базу данных для хранения данных на сервере, напиши вэб приложение для ведения заметок по методу Нилоласа Лумана. Основной функционал: приложение предоставляет пользователю два окна — в одном окне пользователь может писать и редактировать текст. В другом окне пользователь видит древовидны список уже сделаных заметок. Каждой новой заметке автоматчески присваивается обозначение и название заданное пользвателем. Для заметок верхнего уровня идет простая нумерация от 1. Для заметок следующего уровня идет нумерация типа 1.1 далее 1.1.1 и так далее. При созданиее заметки пользователь указывает — это заметка верхнего уровня или указывает родителя заметки. Нужно реализовать функциональность делать ссылки из текста текущей заметки на другие заметки любого уровня в древовидном списке заметок.  Укажи внизу диалогового окна версию приложения

Добавь возможнось выбирать тип заметки - заметка, библиографиеская заметка. Библиографические заметки и простые заметки нумеруй отдельно. Таким образом в окне с древовидным списком будет два списка - линейный библиографический список источников и древовидный список заметок.

Для первого подхода все работает хорошо. Заметки обычные и библиогарфические создаются их возможно редактировать. Нужно сохранить этот функционал и добавить возможноть внутри заметки делать ссылку на запись из библиографического списка или другую заметку. Измени младшую версию. 

В окнах обычных и библиографических заметок не отображаются ранее введеные в базу заметки и блиблиография после перезапуска сервера. Исправь ошибку.
